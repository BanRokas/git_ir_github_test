Git ir Github

Git'as - kodo versijavimo programa.
GitHub'as - versijuoto kodo talpykla internete.

Git komandos

  • init - sukurti git repozitoriją dabartiniame aplankale
  • status - pateikia git repozitorijos dabartinę informaciją
  • add - prideda nurodytus failus prie commit'inmui paruoštų failų (stage'ina)
  • rm --cached || restore --staged - išima nurodytus failus ir stage'inimo zonos (unstage'ina)
  • commit -m "žinutė apie commit'ą" - sukuria kodo versiją iš stage'intų failų ir prideda žinutę apie pakitimus (žinutė turėtų būti trumpa ir aiški)
  • restore - sugrąžina nurodytą/us failą/us į pradinę būseną (jeigu tie failai nebuvo add'inti į stage'ingo zoną)
  • push - nusiunčia lokalios repozitorijos commit'us į išorinę repozitoriją
  • pull - atsiunčia išorinės repozitorijos commit'us į lokalią repozitoriją
  • log - išveda commit'ų sąrašą
  • clone - nuklonuoja nurodytą projektą į dabartinį aplankalą
  • checkout commitID|branchName - pereinama į nurodytą kodo versiją arba kitą šaką
  • branch naujosSakosPavadinimas - sukuriama nauja šaka
  • ...
